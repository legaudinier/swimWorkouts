
# Swim Workout Generator

## Documentation
- Powered by React, and a very unclean typescript that will be cleaned up tbd
- Using Express.js as bare bones server, the database is a spreadsheet
- Export to excel using excel.js

## Available Workouts 
- Distance
- Easy
- Sprint
- Threshold
